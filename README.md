# 🛒 Sales Analysis Project — SQL

## 📌 Project Overview
Analyzed sales data for an e-commerce business to identify
revenue trends, top customers, and product performance.

## 🎯 Business Questions Answered
- Which products generate the most revenue?
- Who are our top spending customers?
- Which months had the highest sales?
- How can we segment customers by spending behavior?
- Which country generates the most revenue?

## 🛠️ Tools Used
- SQL (SQLite)
- sqliteonline.com

## 📊 Key Findings
- Laptop is the top product with $3,200 revenue
- Electronics category dominates with $5,200 total revenue
- UK is the highest revenue market at $3,450
- Alice and Eva are VIP customers (spent above average $1,390)
- March 2024 was the best performing month ($2,550)

## 💡 SQL Concepts Used
- JOINs (single and multiple)
- GROUP BY, ORDER BY, HAVING
- Subqueries
- CTEs (WITH clause)
- Window Functions (RANK, PARTITION BY)
- Running Totals (SUM OVER)
- CASE WHEN customer segmentation
- Date functions (strftime)
